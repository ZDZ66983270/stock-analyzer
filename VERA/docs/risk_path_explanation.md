# 风险生命周期路径释义 (D0-D6)

为了帮助您理解 VERA 的“风险路径状态机”，我们通过以下生命周期曲线来直观图示价格所处的不同风险阶段。

## 1. 轨迹映射逻辑
在历史轨迹图中，纵轴代表您的资产在“回撤-修复”周期中的相对位置。我们对 Y 轴进行了重构，使其更符合直觉：**位置越高，状态越稳；位置越低，风险越深。**

| 状态 | 语义 | 轴高度 | 价格位置解读 |
| :--- | :--- | :--- | :--- |
| **D0** | **稳定高位** | **6 (最高)** | 资产处于历史高点附近，路径极其稳健。 |
| **D6** | **完全修复** | **5** | 经历过大跌后，价格已几乎涨回到前高。 |
| **D1** | **浅大回撤** | **4** | 开始从高位回落（10%左右），风险早期。 |
| **D5** | **修复中段** | **3** | 价格从谷底反弹过半，进入修复中段。 |
| **D2** | **中度回撤** | **2** | 回撤加深（20%以上），风险进一步放大。 |
| **D4** | **反弹早期** | **1** | 刚从谷底爬起，极易由于基础不稳而“二次探底”。 |
| **D3** | **深度回撤** | **0 (最低)** | 价格处于大级别的“黄金坑”或“深跌中心”。 |

## 2. 案例分析：中信银行 (601998.SH)
如果您看到轨迹从中信银行 **D6 -> D5 -> D4** 下行，其背后的风险语义是：

1. **完全修复 (D6)**: 资产曾尝试挑战前期高点。
2. **修复中段 (D5)**: 随后价格无力维系，开始回落至修复的中程。
3. **反弹早期 (D4)**: 跌势继续转弱，回落至刚脱离底部的反弹起点。

> [!IMPORTANT]
> **视觉下行 = 状态恶化**。
> 这种轨迹图直观展示了资产正在经历一个**“修复失败 / 二次探底”**的过程。它告诉您之前的修复努力正在被抵消，资产正在重新滑向底部的风险区（D3）。

## 3. 如何阅读此图
- **向上爬升**：代表资产正在走出泥潭，风险正在修复。
- **向下俯冲**：代表资产正在陷入深渊，或者之前的反弹正在瓦解。
- **高位横盘**：代表资产极其稳健（D0）。

---
*注：轨迹图描述的是历史价格路径的位置属性，旨在量化资产当下的“抗跌/修复”成色，不构成未来涨跌的预测信号。*
